### Requirements:
- Make sure you are using node version 16.15.0

1. Install the packages:
```sh
npm install --legacy-peer-deps
```
2. Create a .env file:
```sh
cp .env.example .env
```
3. Start coding! and the webpack dev server with live reload, for windows, mac, linux or Gitpod:

```bash
npm run start
```
