04 Unix Folder Structure

  • /
    • /bin (Binaries)
    • /boot (Boot)
    • /dev (Devices)
    • /etc (Configuration)
    • /home (Home)
      • /user1
      • /user2
    • /lib (Libraries)
    • /media (Media)
    • /mnt (Mount)
    • /opt (Optional)
    • /proc (Process)
    • /root (Superuser root)
    • /run (Runtime)
    • /sbin (Superuser binary)
    • /srv (Services)
    • /tmp (Temporary)
    • /usr (User)
    • /var (Variable)

/bin: Contains executable files (programs) that are essential for the system's operation and used by all system users.

/boot: Contains files related to the system's boot process.

/dev: Device files. In Unix, everything is a file, and devices are also represented as files in the /dev directory.

/etc: Contains system and application configuration files.

/home: User home directories. Each user has their own subdirectory in /home.

/lib: Contains shared libraries used by programs in /bin and /sbin.

/media: Mount point for removable media such as USB, DVD, etc.

/mnt: Temporary mount point for additional file systems.

/opt: Contains additional software packages that are not part of the main OS distribution.

/proc: Contains information about running processes.

/root: Home directory for the root user (superuser).

/run: Contains temporary file data that is valid only during system runtime.

/sbin: Contains essential binary files for system administration, accessible only by the superuser.

/srv: Contains data specific to services provided by the system.

/tmp: Stores temporary files.

/usr: Contains files and programs for users. It can be shared among systems and remotely mounted.

/var: Stores files that can dynamically change in size, such as logs, temporary files, and databases.
