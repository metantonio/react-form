09 Directory Creation

We can create new folders with the mkdir command, a short form of "make directory".

It is common to create folders in the current directory, with the following structure mkdir new_folder, where new_folder is the name of the new folder.

But it is also possible, and indeed quite useful, to specify where the folder will be created without having to move from our current directory, using relative paths.
