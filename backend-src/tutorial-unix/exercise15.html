15 Múltiples Comandos Condionados

Es posible ejecutar varios comandos múltiples y que éstos se ejecuten dependiendo, o no, de la ejecución exitósa de los comandos previos

  • && Operador: Ejecuta el siguiente comando sólo si el anterior fue ejecutado exitósamente.
  • || Operador: Ejecuta el siguiente comando sólo si el anterior fue ejecutado fallidamente.
pwd && ls

📝 Instrucciones:

Coloca el(los) operador(es) correcto para que se pueda ejecutar el siguiente comando

man && ls || cat hello.txt

Cambia únicamente los operadores.
