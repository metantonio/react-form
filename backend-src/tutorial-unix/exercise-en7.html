07 File Search

It's common to encounter directories with thousands of files and dozens of subdirectories, making it necessary to filter the search or obtain a specific file. For this, we use the find command.

The find command has a large number of parameters.

To view the full list of arguments, use find --help.

The most basic structure to recursively search for a file from the current directory is find ./ -type f -name "name-of-file.txt", where the argument "-type f" indicates that we want to find files.
