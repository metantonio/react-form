03 Navigate Between Directories

To move between directories, we can use the cd (current directory) command and then provide the path to where we want to go.

The path can be absolute (specifying the complete address)

cd C:/users/documents/images/

It can also be relative. Suppose we are currently in the 'documents' directory and want to go to the 'images' directory.

cd ./images

Now that we are in the 'images' directory, let's say we want to go back 1 level to return to the 'documents' directory.

cd ../
