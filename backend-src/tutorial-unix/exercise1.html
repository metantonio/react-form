01 Hello UNIX

La consola de Unix, también conocida como terminal o shell, es una interfaz de línea de comandos que permite a los usuarios interactuar con el sistema operativo Unix o sistemas operativos basados en Unix, como Linux y macOS, a través de comandos de texto.

En este tutorial simularemos tener la consola de UNIX, pero los comandos los escribiremos en la sección inferior. Para ejecutar el código presionaremos la tecla ENTER
