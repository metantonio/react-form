05 File Copy

From the terminal, we can copy files using the cp command.

The cp command requires two arguments, which can be paths (absolute and/or relative). The first argument indicates where to copy from, and the second argument indicates where to place the copied content.

When paths are not specified, and only names are used, it is understood that the terminal is currently located in the same directory as the file to be copied, and its copy will be in the same place.

cp original.txt copy.txt
