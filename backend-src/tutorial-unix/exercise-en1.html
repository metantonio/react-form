01 Hello UNIX

The Unix console, also known as the terminal or shell, is a command-line interface that allows users to interact with the Unix operating system or Unix-based operating systems such as Linux and macOS through text commands.

In this tutorial, we will simulate having the UNIX console, but we will write the commands in the lower section. To execute the code, press the ENTER key.
