11 View Document Content

We can safely view the content of any text file with the commands cat and less.

When the file is short, it is recommended to use cat file_name, but when the file is longer, it can be inconvenient, so it is recommended to use less file_name

It is also possible, and in fact quite useful, to specify the file to be read with relative paths.
