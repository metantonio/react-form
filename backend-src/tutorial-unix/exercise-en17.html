17 Inodes

An inode is a concept that describes an object in the file system, such as a directory or a file.

An inode contains metadata about the attributes and location on the hard disk of the object. In Unix-based systems, you can represent an inode as a number (pointer).

To reveal the inode of a file or directory, you can simply use:

ls -i
