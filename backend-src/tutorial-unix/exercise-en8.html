08 The PIPE Operator

The UNIX console allows us to concatenate code, and we can do this thanks to the PIPE operator |. It is called this way because when placed twice, it simulates a pipe "||".

It is common to see the pipe operator associated with the grep command, which is usually helpful for filtering patterns.

The most basic structure to recursively search for file types from the current directory and specify a pattern is find ./ -type f -name "*.extension" | grep "pattern-to-search".

Where the argument "-type f" indicates that we want to find files.
