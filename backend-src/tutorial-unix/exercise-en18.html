18 Hard Link and Soft Link

The ln command is used to create a link to a file. It's similar to creating shortcuts in Windows.

By default, the ln command creates "hard links" to files. This means that if the original file is moved or renamed, we can still access it because the "hard link" links it through the inode.

On the other hand, the ln -s command creates "soft links" to files. It's a direct pointer to the file and not to the original inode (creating a new inode), so if the file is renamed or moved, even if the "soft link" exists, we no longer have the correct path to the original file.

To create these links, it's necessary to first navigate to the folder where we want to create this "shortcut" and then provide the path to the file using:

ln ./path-to-file/or-folder
