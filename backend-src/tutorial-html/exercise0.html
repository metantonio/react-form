Fundamentos:

Hemos preparado estos ejercicios para cualquier persona interesada en aprender HTML desde cero. Durante este curso, repasaremos los siguientes conceptos:

  • Familiarizarse con la mayoría de las etiquetas HTML más populares.
  • Usar varias etiquetas juntas para crear un sitio web HTML simple.
  • Vinculación de sitios web mediante enlaces (anchors) con rutas relativas o absolutas.
  • La etiqueta HTML de audio y video.
