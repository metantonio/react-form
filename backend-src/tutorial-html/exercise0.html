Fundamentals:

We have prepared these exercises for anyone interested in learning HTML from scratch. During this course, we will cover the following concepts:

  • Familiarizing yourself with most popular HTML tags.
  • Using multiple tags together to create a simple HTML website.
  • Linking websites through links (anchors) with relative or absolute paths.
  • The HTML audio and video tags.
