01 The Console

Te damos la bienvenida al tutorial de Python.

La consola de Python es una herramienta fundamental para desarrolladores. Permite imprimir mensajes y depurar código. Una de las funciones más comunes es print()

📝 Instrucciones:

Elimina todo el contenido actual en la sección de código, y copia la siguiente línea de código:


print("Hello World");

Observa la salida en la consola del navegador y verifica si aparece el mensaje "Hello World". ¡Haz clic en "Run Code" para ver el resultado! y luego en "Check" para verificar la validez de tu respuesta.
