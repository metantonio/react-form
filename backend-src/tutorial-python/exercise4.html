04 Lenguaje de Tipado Fuerte

Python es un lenguaje de programación de tipado dinámico, lo que significa que el tipo de una variable se determina en tiempo de ejecución y puede cambiar durante la ejecución de un programa.

Python también se considera un lenguaje de tipado fuerte porque no permite la conversión implícita de tipo entre diferentes tipos de variables. Por ejemplo, no puedes sumar una cadena a un entero sin convertir explícitamente uno de ellos al otro tipo.

     

sum = None


print(sum)
