20 For Loop anidados

Una tarea habitual en programación, es tener que encontrar todas las posibles combinaciones que se pueden obtener, dados una serie de elementos.

Y la forma más sencilla de hacerlo es con ciclos For Loop anidados (uno dentro de otro).

Podemos tener ciclos anidados de la siguiente manera:

  

for(let i=0; i < 3; i=i+1){
    for(let j=0; j < 3; j=j+1){
        console.log(`value i: ${i}, value j: ${j}`);
    }
}

Una manera rápida de saber el total de combinaciones posibles es multiplicar la cantidad de ciclos en cada loop, en este caso serían: 3 * 3 = 9 combinaciones posibles.

📝 Instrucciones:

A continuación, elimina todo el contenido en la sección de código y copia lo siguiente:


     
let movies = ["Matrix", "Toy Story", "The Incredibles", "Up"];

Encuentra todas las combinaciones posibles con la estructura movies + description + finish
