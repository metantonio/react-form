09 Creación de Funciones

Para entender las funciones es necesario ver cómo se construyen.

La construcción de una función pasa por definirla primero, ya sea de forma general o en su forma de función flecha:


//general function 
function nameOfFunction(){
    //logic of function 
    }
    

//arrow function 
const nameOfFunction = () => {
    //logic of function    
    }

Imagina que quieres crear una función que reciba dos valores, los sume y devuelva el resultado para que pueda ser almacenado en una variable, necesitas conocer:

  • Parámetros
  • Return

Veamos cómo sería:


//general function 
function addition(parameter1, parameter2){
    //logic of function 
    return parameter1 + parameter2;
    }

let result = addition(1,2);
console.log(result);
    

//arrow function 
const addition = (parameter1, parameter2) => {
    //logic of function 
    return parameter1 + parameter2;   
    }

let result = addition(1,2);
console.log(result);
