09 Function Creation

To understand functions, it is necessary to see how they are constructed.

The construction of a function involves defining it first, either in a general form or in its arrow function form:


//general function 
function nameOfFunction(){
    //logic of function 
}


//arrow function 
const nameOfFunction = () => {
    //logic of function    
}

Imagine you want to create a function that takes two values, adds them, and returns the result to be stored in a variable. You need to know:

  • Parameters
  • Return

Let's see how it would be:


//general function 
function addition(parameter1, parameter2){
    //logic of function 
    return parameter1 + parameter2;
}

let result = addition(1,2);
console.log(result);


//arrow function 
const addition = (parameter1, parameter2) => {
    //logic of function 
    return parameter1 + parameter2;   
}

let result = addition(1,2);
console.log(result);
