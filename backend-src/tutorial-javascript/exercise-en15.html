15 Random Numbers

In JavaScript, there is an internal library called Math that allows us to, among other things, obtain random numbers.

The way to obtain random numbers is with Math.random()


console.log(Math.random()); //returns a random number between 0 and 1, excluding 1.

As we saw, we can return numbers between 0 - 1, excluding 1.

If we want to increase the range of returned numbers, for example, up to 5 excluding it:


console.log(Math.random() * 5); //returns a random decimal between 0 and 5, excluding 5.

As 5 is not included, we see that the numbers actually range from 0 to 4.9999999

Now, suppose we only want to obtain whole numbers from the previous range, we use Math.floor():


console.log(Math.floor((Math.random() * 5))); //returns a random whole number between 0 and 5, excluding 5.
