04 Lenguaje de Tipado Débil

Javascript es un lenguaje de tipado débil, esto significa que una variable que sea del tipo 'cadena de texto', puede después convertirse en una variable que contenga números (enteros, flotantes, etc...), y en otros tipos de datos en cualquier momento del código.

No sólo eso, también podemos declarar variables sin tener que asignarles un valor en principio, prueba el siguiente código por ejemplo:

let suma;
console.log(suma);
