14 Switch for Comparisons?

Although it's not a good practice as it can lead to data type errors, among other issues; you could use the switch() to perform value comparisons.

The way to do it would be by passing true to the switch() function


switch(true) {
    case > 99:
        //logic
        break;
    //Other cases
    default:
        //logic
        break;
}

In the case of these switch(true), if numerical values are being evaluated, it is strongly recommended to start from the case with the highest numerical value to the lowest, in order.
