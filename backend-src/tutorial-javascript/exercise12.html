12 Simulando Puerta lógicas

Ya sabemos que podemos comparar las condiciones con:

  • Igualdad: ==
  • Igualdad de valor y de tipo de dato: ===
  • Mayor que: >
  • Mayor o igual que: >=
  • Menor que: <
  • Menor o igual que: <=

También podemos enlazar varias condiciones de forma incluyente y/o excluyente. Ejemplo:

  • Ambas condiciones se deben cumplir (AND) condition1 && condition2
  • Al menos una de la condiciones se debe cumplir (OR) condition1 || condition2
