18 Arreglos

En javascript tenemos algunas estucturas de datos, entre ellas los Arreglos (arrays)

Un array es una estructura de datos que se utiliza para almacenar una colección ordenada de elementos. Estos elementos pueden ser de cualquier tipo, como números, cadenas de texto, booleanos, objetos, funciones, u otros arrays, lo que permite crear estructuras de datos complejas y flexibles.

Declaración de un Array:

 
let myArray = ["Hello", 1, true, ["Another array inside"]];
console.log(myArray.length);

Índices de un Array:

 
let myArray = [10, 20, 30];
console.log(myArray[0]); // Imprime 10 
console.log(myArray[1]); // Imprime 20
console.log(myArray[2]); // Imprime 30

Longitud de un Array:

 
let myArray = [10, 20, 30];
console.log(myArray.length); // Imprime 3, porque tiene 3 elementos

Manipulación de un Array:

 
let myArray = [10, 20, 30];
myArray.push(40) // se agrega un elemento nuevo al final del array 
console.log(myArray); // imprime [10, 20, 30, 40] 
myArray.pop() // elimina el último elemento del array 
console.log(myArray); // el array vuelve a ser [10, 20, 30]
