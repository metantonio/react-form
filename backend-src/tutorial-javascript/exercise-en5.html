05 Declaration of Constant Variables

Javascript allows the declaration of 'constant' variables, which means that once a value is assigned, the value of the constant cannot be changed in any way.

Here is an example of declaring the constant variable PI (by convention, constants are written in uppercase):

       

const PI = 3.1415;
console.log(PI);
