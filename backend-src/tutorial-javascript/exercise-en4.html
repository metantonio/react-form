04 Weakly Typed Language

Javascript is a weakly typed language, which means that a variable of type 'string' can later become a variable that holds numbers (integers, floats, etc.), and other data types at any point in the code.

Not only that, we can also declare variables without having to assign them a value initially. Try the following code, for example:

     

let sum;


console.log(sum);
