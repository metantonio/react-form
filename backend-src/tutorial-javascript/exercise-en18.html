18 Arrays

In JavaScript, we have some data structures, among them, Arrays.

An array is a data structure used to store an ordered collection of elements. These elements can be of any type, such as numbers, strings, booleans, objects, functions, or other arrays, allowing the creation of complex and flexible data structures.

Declaration of an Array:

 
let myArray = ["Hello", 1, true, ["Another array inside"]];
console.log(myArray);

Indexes of an Array:

 
let myArray = [10, 20, 30];
console.log(myArray[0]); // Prints 10 
console.log(myArray[1]); // Prints 20
console.log(myArray[2]); // Prints 30

Length of an Array:

 
let myArray = [10, 20, 30];
console.log(myArray.length); // Prints 3, because it has 3 elements

Manipulation of an Array:

 
let myArray = [10, 20, 30];
myArray.push(40) // Adds a new element to the end of the array 
console.log(myArray); // Prints [10, 20, 30, 40] 
myArray.pop() // Removes the last element from the array 
console.log(myArray); // The array is again [10, 20, 30]
