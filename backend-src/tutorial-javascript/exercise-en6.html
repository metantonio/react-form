06 Scope of Declaration

let was introduced as an alternative to var for declaring variables. Unlike var, variables declared with let have a stricter block scope.

This means that variables declared with let only exist within the curly braces { }.

Observe the following examples:

        
function exampleVar() {
    if (true) {
        var x = 10;            
        console.log(`Inside if block (var): ${x}`);                
    }        
    console.log(`Outside if block (var): ${x}`);      
}
          
exampleVar();
        
        

function exampleLet() {    
    if (true) {     
        var x = 10;        
        console.log(`Inside if block (let): ${x}`);           
    }      
    //Esto generar√° un error, ya que 'x' no est√° definido fuera del bloque if        
    //console.log(`Outside if block (let): ${x}`);      
}        
exampleLet();
