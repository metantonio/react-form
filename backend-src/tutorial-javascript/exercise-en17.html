17 For + If

Within loops, it's common to evaluate conditions.

In the following example, certain logic is executed only when the loop variable i is equal to 3:

 
for(let i=0; i<5; i=i+1){
    if(i==3){
        console.log('Ejecutando');
        //logic here 
    }else{
        console.log('error');
    }     
}

Try the above code in the terminal.
