13 El Switch

En ocasiones, podríamos tener una serie de if...else if anidados, lo que puede hacer el código poco legible.

En javascript tenemos la función switch() la cuál tiene la siguiente estructura:

 
switch(value) => {
    case "some-condition":
        //logic 
        break;
    case "some-condition2":
        //logic 
        break;
    default:
        //logic 
        break;
}

📝 Instrucciones:

Imagina que tienes un sistema de puntuación que va desde el 0 (cero) hasta el 4 (cuatro). Para cada uno de los puntajes hay un color relacionado:

  • Para el puntaje 0, el color devuelto debe ser: "black"
  • Para el puntaje 1, el color devuelto debe ser: "yellow"
  • Para el puntaje 2, el color devuelto debe ser: "orange"
  • Para el puntaje 3, el color devuelto debe ser: "red"
  • Para el puntaje 4, el color devuelto debe ser: "green"
  • Para cualquier otro valor, el color devuelto debe ser: "blue"

A continuación, elimina todo el contenido en la sección de código y copia lo siguiente:
