08 Explorando Funciones en JavaScript

¡Las funciones son una herramienta poderosa para los programadores!

Imagina las funciones como cajas negras: proporcionas una entrada y obtienes una salida.

Las fortalezas de las funciones incluyen, entre otras cosas:

  • Encapsulación de código, manteniéndolo oculto al resto del programa. Este es un concepto crucial que profundizarás a medida que avances en tu desarrollo.
  • Reutilización de código en diferentes programas, tratando las funciones como herramientas en una caja de herramientas. Este enfoque mejora la eficiencia y la limpieza del código.

Ya existe una función llamada isEven() que devuelve true para números pares y false para números impares cuando se pasa como argumento.

A continuación, elimina todo el contenido en la sección de código y copia lo siguiente:

    console.log("ejecutando la función");
    if(someValue % 2 == 0){
        console.log(`es Par, retorna true`);
        return true;
        }
    else{
        console.log(`es Impar, retorna false`);
        return false;
        }
    }
