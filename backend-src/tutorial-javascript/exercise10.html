10 Completa la función

📝 Instrucciones:

A continuación, se te propone una función que hace uso de "function strings" o "f-strings" para concatenar.


function fString(name, year){
    let newString = `My name is ${name} and have ${year} old. I'm from`;
    return newString;
    }

console.log(fString('John', 30, "London"));
console.log(fString('Mary', 24, "New York"));
console.log(fString('David', 42, "California"));

Modifica la funciónfString() para que pueda recibir los argumentos pasados, y pueda retornar lo siguiente:

My name is John and have 30 old. I'm from London My name is Mary and have 24 old. I'm from New York My name is David and have 42 old. I'm from California

Pista:

No olvides el punto y coma (;) al final de cada línea de código
