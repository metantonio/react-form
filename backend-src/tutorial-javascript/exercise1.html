01 The Console

Te damos la bienvenida al tutorial de JavaScript.

La consola de JavaScript es una herramienta fundamental para desarrolladores. Permite imprimir mensajes y depurar código. Una de las funciones más comunes es console.log().

📝 Instrucciones:

Elimina todo el contenido actual en la sección de código, presiona el botón de Ejemplo y usa la siguiente línea de código:

```javascript console.log("Hello World"); ```

Observa la salida en la consola del navegador y verifica si aparece el mensaje "Hello World". ¡Haz clic en "Ejecutar" para ver el resultado!
