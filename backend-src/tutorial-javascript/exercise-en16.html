16 For Loop

In most programming languages, there are ways to iterate, which means repeating a procedure a certain number of times.

The way to iterate in Javascript is with the for() loop.

Let's look at the classic structure of the For Loop:

 
for(let i=0; i<5; i=i+1){ //i starts at 0, as long as i is less than 5, the increment of i will be its previous value plus one 
    console.log(i);
}
