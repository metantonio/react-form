17 For + If

Dentro de los loops es normal evualar la condiciones.

En el siguiente ejemplo, sólo cuando el ciclo i sea igual a 3, es que se ejecuta cierta lógica:

 
for(let i=0; i<5; i++){
    if(i==3){
        console.log('Ejecutando')
        //logic here 
    }else{
        console.log('error')
    }     
}

Prueba el código anterior en la terminal

📝 Instrucciones:

A continuación, elimina todo el contenido en la sección de código y copia lo siguiente:


for(let i=0; i<5; i=i+1){ 
    console.log(i)
}

Modifica el For Loop de modo que el ciclo vaya desde 0 hasta 21 (sin incluir el 21), y además ocurra lo siguiente:

  • Cuando la variable i sea par, se debe imprimir en consola: "I'm even"
  • Cuando la variable i sea impar, se debe imprimir en consola: "I'm odd"
