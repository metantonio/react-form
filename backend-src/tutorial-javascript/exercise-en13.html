13 The Switch

Sometimes, we might have a series of nested if...else if statements, which can make the code less readable.

In JavaScript, we have the switch() statement, which has the following structure:

 
    
switch(value) {
    case "some-condition":
        //logic
        break;
    case "some-condition2":
        //logic
        break;
    default:
        //logic
        break;
}
