05 Declaración de Variables Constantes

Javascript permite la declaración de variables del tipo 'constante', esto significa que una vez se le asigna valor, no se puede cambiar de ninguna manera el valor de la constante

Aquí tenemos un ejemplo de declaración de la constante PI (por convención las constante se escriben en mayúsculas):

       

const PI = 3.1415;
console.log(PI);
