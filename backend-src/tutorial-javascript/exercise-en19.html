19 Looping Through Arrays with For Loop

Something quite common is looping through arrays, and we have ways to do it in JavaScript. Perhaps the most generic and powerful way is with the generic For Loop.

If you haven't noticed, the index of the last element in an array will be equal to the array's length minus 1.

We can loop through an array as follows:

 
let myArray = ["Hello", 1, true, false];

for(let i=0; i < myArray.length; i++){
    console.log(myArray[i]);
}

Even though the above code can run, it's not efficient because every time it goes through the array, it has to get the length of the array with myArray.length. And this is particularly terrible when the array has thousands or millions of elements.

Therefore, it's better to store the array's length, so this reading is done only once:

 
let myArray = ["Hello", 1, true, false];
let myArrayLength = myArray.length;

for(let i=0; i < myArrayLength; i++){
    console.log(myArray[i]);
}

Try the above code in the terminal
