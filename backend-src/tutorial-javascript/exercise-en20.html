20 Nested For Loops

A common task in programming is to find all possible combinations that can be obtained given a series of elements.

The simplest way to do this is with nested For Loops (one inside the other).

We can have nested loops as follows:


 
for(let i=0; i < 3; i=i+1){
    for(let j=0; j < 3; j=j+1){
        console.log(`value i: ${i}, value j: ${j}`);
    }
}

A quick way to know the total possible combinations is to multiply the number of cycles in each loop, in this case, it would be: 3 * 3 = 9 possible combinations.
